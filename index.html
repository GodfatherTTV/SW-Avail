1. Quick & Easy: Google Sheets + Discord

Firebase rules are one big JSON object, so you can’t just “add below”. If you leave both, you’ll get a syntax error because you can’t have duplicate keys.

What to do:

  1. Firebase Console → Realtime Database → Rules tab
  2. Delete everything in the editor
  3. Paste this:
JSON
json
{  "rules": {    "orgs": {      "sw-avail": {        ".read": true,        ".write": true      }    }  }}
  1. Hit Publish

What that does:

  • Only the /orgs/sw-avail path is open read/write
  • Everything else in your database stays locked down by default
  • Anybody with the link can view/edit the calendar — perfect for 20 trusted org members
